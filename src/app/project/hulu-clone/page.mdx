The Hulu Clone is another UI focused project. It accesses <a href='https://www.themoviedb.org/?language=en-US' target='_blank'>TMDB</a>'s API to pull in a list of trending movies. <b>PLEASE NOTE</b>
I did pass in a filter for Non-Adult movies, but I cannot control what ends up coming through the API.

<Image src={huluHome} alt=''/>

Once the data is pulled in from the API, it is displayed in a flexbox that grows depending on the size of the screen. The project boasts lazy loading images as well as cached routes. You can click
on the different category of movies which will then refine the displayed movies to the specific category.
