The lyric finder application is a simple react application that pulls music data from a <a href='https://genius.com/' target='_blank'>Genius</a> API. Initially, the each song in the "Top Charts"
category from the API is listed. Each song can be clicked on and you will be redirected to the lyrics on the Genius website.

A user can also search for any albums, artists and songs which will then refetch the lyrics with the search parameters.
