This React application was a way to show off several different sorting algorithms in a visual way. A random array is generated and each number in the array is represented by a bar. You then have the 
option to sort the array in four different ways

- Bubble Sort
- Selection Sort
- Quick Sort
- Insertion Sort

<Image src={sortHome} alt=''/>

Once a sorting method has been chosen, each bar will be sorted from shortest to longest based off of it's numerical value. Once the array is sorted, each bar will turn green. Once the array is sorted,
you can then re-randomize the array to try a different sorting method.
