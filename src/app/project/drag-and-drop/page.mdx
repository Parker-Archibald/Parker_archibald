The Drag and Drop project is a simple Next.js application utilizing <a href='https://github.com/atlassian/react-beautiful-dnd' target='_blank'>react-beautiful-dnd</a> to manage dragging and dropping tasks
into different groups. This simple kanban style task manager allows you to drag tasks between columns to track the status of each task - either To Do, In Progress or Done.

<Image src={dragndropHome} alt=''/>

Adding a task to the <a href='https://firebase.google.com/' target='_blank'>Firebase</a> database is fairly simple, click on the green plus icon in any column to open up the modal where you can choose 
which column the task should belong to and add an image to the task. You can also search for tasks which will filter out any tasks that does not fit within the searched input. Deleting any task is
as basic as clicking the red x next to the task.
